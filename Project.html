MEFASH Generating System

MEFASH is a Fashion Generating System for men.

  • BACKGROUND
    • The fashion industry is divided into CREATIVE and SALES FUNCTION (i.e. Design and production on one hand, sales and distribution on the other). This project and system will be focusing on the Creative/Design aspect of the fashion industry. One of the problems in the fashion industry in Nigeria which the system will be addressing in this project, is the problem of coming up with a new style or design. As the fashion world changes an improve there should be avenue for new ideas to spring up such as a fashion design generating system.

  • AIM/OBJECTIVE OF THE PROJECT
    • AIM: The aim of this project is to build a system that can generate new fashion designs for men. OBJECTIVES ARE:
  • 1. To first identify the correct Machine Learning Algorithm to be used.
  • 2. Build a Machine Learning model that can generate new menâ€™s fashion designs.

  • METHODOLOGY
    • The system will follow the Hypothesis-driven approach of data analytics and the type of machine learning algorithm that will be used is a supervised learning algorithm, this is because the system will be supplied both the Training and Testing Dataset that it will be making use of. The dataset will include collection of images of previous Nigerian fashion designs (using ICRAWLER to scrap images from the web), which will be used to train the model. The system will be using the Deep Convolutional Generative Adversarial Network (DCGAN) model for image processing and classification.

  • EXPECTED RESULT
    • The expected result of this project is a working system that is able to generate new styles and designs for a particular person.
